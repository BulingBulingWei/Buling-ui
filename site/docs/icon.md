## 安装

目前仅有 alpha 版本可安装

```bash
npm i @buling-ui-icon
```

## 使用示例

```
<template>
  <AngleDownIcon color="#0052d9" size="0.36rem"/>
</template>

<script setup>
import { AngleDownIcon } from '@buling-ui-icon'
</script>
```
